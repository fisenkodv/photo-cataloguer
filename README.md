# Photo Cataloguer

## Create Environment
 - Install *virtualenv*
 ```
 $ [sudo] pip install virtualenv
 ```
 - Create an environment
```
$ virtualenv ENV
```
- Activate script
```
$ source [ENV]/bin/activate
``` 
- Install dependencies
```
$ pip install -r requirements.txt 
```

- Deactivate script
```
$ deactivate
```
